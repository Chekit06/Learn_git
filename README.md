# LazyGit (Interactive UI to manage git)
-   ?     -  to view all available shortcut keys
-   space -  staging a particular file
-   a     -  stage all
-   p     -  pull latest changes
-   P     -  Push latest commits
-   n     -  new branch
- shift+m -  merging feature branch into main branch
